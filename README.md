# Night Owls Detector

the script shows a list of students who have submitted assignments to be checked at night

# Installation
```bash 
pip install -r requirements.txt # alternatively try pip3
``` 
# Usage
```bash 
python3 seek_dev_nighters.py 
Devmat midnighters list:
name1
name2
```
